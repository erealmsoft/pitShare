## case

```javascript
var arr = [9 , 10, 14, 12]
arr.sort();  // [ 10, 12, 14, 9 ]
        // expect [ 9, 10, 12, 14 ]
```

## solution

```
var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a-b});

```

The result of points will be:

```
1,5,10,25,40,100
```
