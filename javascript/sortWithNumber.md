## case

```javascript
var arr = [9 , 10, 14, 12]
arr.sort();  // [9, 14, 12, 10]
        // expect [14, 12, 10, 9]
```

## solution

```
var points = [40, 100, 1, 5, 25, 10];
points.sort(function(a, b){return a-b});

```

The result of points will be:

```
1,5,10,25,40,100
```
